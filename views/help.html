Help

In this help section you will understand each part of the website. This website online represents data of a .csv file and it can show different statistics extracted from this file. The website has three sections:

  1. Start: the beginning page only inform the user about the project.
  2. Statistics: section that shows the statistics and results of the derbies and the next matches. In the seasons subsection you can select an specific season to check de results.
  3. Help: help section.
