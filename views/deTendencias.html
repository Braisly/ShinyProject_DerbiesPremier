Datos Estadísticos - Tendencias

En esta sección de tendencias se muestran los coeficientes necesarios para calcular la recta de regresión simple. Esta recta de regresión se realiza con el comando tslm, con el cuál es posible realizarla con estacionalidad, dando lugar a esta fórmul: Yt = a + bt + cn, dónde b representa m1 que sería la pendiente, o tendencia. Y habría que añadir cn siendo n el número de mes (superior a Enero) y c el coeficiente asociado a ese mes.
