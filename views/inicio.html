SciDB-R Project

Análisis espacio-temporal de variables geofísicas multidimensionales

El objetivo del proyecto cómo muy bien se indica es el de realizar un análisis epacio-temporal de variables geofísicas multidimensionales, empleando tecnologías novedosas y que pudiesen aportar y aumentar el conocimiento de las mismas. Se empeleará como base de datos SciDB, una base de datos de arrays. A la hora de realizar todo el procesamiento, análisis y cálculo de datos se empleará el lenguaje de programación estadístico R. Para presentar los datos de manera visual y que sean atractivos para el usuario, se utilizará el framework Shiny para crear aplicaciones web a través de R. En este caso podemos ver, la página web con cuatro menús principales: Inicio, Gráficos, Datos Estadísticos y Ayuda. En el menú Ayuda se explicarán las dudas que puedan surgir y la funcionalidad que tiene cada menú.
